13. 두개의 List 교집합, 차집합, 합집합, 부분집합

List<Integer> A = new ArrayList<>();
List<Integer> B = new ArrayList<>();

## 교집합
A.retainAll(B); // A와 B의 교집합

## 부분칩합
A.containsAll(B); // B가 A의 부분집합인가? T or F

## 차집합
A.removeAll(B); // A에서 B를 모두제거

## 합집합
A.addAll(B); // A에 B를 합침 (중복이 있다.)
