5. 람다식 예제

```
public class Main{
public static void main(String[] args){
Goods com = new Goods(){
public void doSome(){
System.out.println("do!!");
}
}

    com.doSome();

}
}
```

-> 람다식 변환

```
public class Main{
public static void main(String[] args){
Goods com = () -> System.out.println("do!!");
com.doSome();
}
}
```

()에는 매개변수를 입력하고 ->를 입력하고 {}안에 실행할 코드를 작성한다.

람다식을 사용하기 위한 인터페이스에는 추상메소드가 단 1개 여야 한다는 조건이 있다.
