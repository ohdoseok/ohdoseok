재정의도 가능

```
public class Product {
    int id;
    String name;
    public Product(int id, String name) {
        this.id = id;
        this.name = name;
        }
    @Override
    public boolean equals(Object object) {
        Product product = (Product) object;         // name은 상관없이, id만 같으면 true를 리턴합니다.
        if (product.id == this.id) {
            return true;
            }
        return false;
        }
}
```
